# Results

This folder stores outputs produced by the RL agent training runs:

| File | Description |
|------|-------------|
| `rewards.npy` | NumPy array of per-episode total rewards (shape: `(n_episodes,)`) |
| `q_table.npy` | Final learned Q-table (shape: `(n_states, n_actions)`) |
| `training_curve.png` | Matplotlib plot of the reward curve |

These files are generated by running:

```bash
python ../code/rl_agent.py
```

> **Note:** Binary result files (`*.npy`, `*.png`) are excluded from version control via `.gitignore`.
> Commit only summary CSVs or final plots that you want to share.
