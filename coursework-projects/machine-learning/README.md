# Machine Learning – Feature Reduction for Drug Design (ECE 503)

## Overview
This project applies Principal Component Analysis (PCA) to reduce the high-dimensional molecular feature space while maintaining predictive performance. The goal is to optimize drug design workflows for Sirtuin6 targets.

## Methods
- Dimensionality reduction using PCA
- Statistical evaluation of model robustness before and after reduction
- Validation to ensure predictive performance is preserved

## System Architecture
1. Data collection and feature extraction
2. Feature scaling and normalization
3. PCA for dimensionality reduction
4. Model evaluation before and after PCA
5. Statistical validation of results

## Technologies
Python  
PCA  
Dimensionality Reduction  
Statistical Modeling  

## Results
- Reduced feature space while maintaining predictive accuracy
- Demonstrated model stability and robustness with fewer dimensions
- Optimized workflow for computational drug discovery

## Future Work
- Explore non-linear dimensionality reduction techniques (e.g., t-SNE, UMAP)
- Integrate PCA with deep learning models for enhanced predictive performance
- Visualize feature importance and correlations
