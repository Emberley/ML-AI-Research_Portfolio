# Coursework Projects

This folder contains implementations and write-ups from academic coursework.

## Structure

Each sub-folder represents a single course or assignment:

```
coursework-projects/
└── <course-name>/
    ├── code/
    ├── notebooks/
    └── README.md
```

## Adding a New Coursework Project

1. Create a sub-folder named after the course (e.g., `deep-learning-fundamentals/`).
2. Add source code in `code/` and Jupyter notebooks in `notebooks/`.
3. Write a short `README.md` describing the assignment, dataset, and key results.
